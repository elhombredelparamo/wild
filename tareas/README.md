# Carpeta de Tareas

## Propósito
Esta carpeta sirve como almacenamiento temporal de creatividad y registro de trabajo pendiente del proyecto Wild.

## Función
- **Registro de ideas:** Anotar conceptos y mejoras futuras
- **Gestión de tareas:** Mantener un listado de trabajo pendiente
- **Organización:** Separar las tareas pendientes del contexto actual del proyecto
- **Priorización:** Ayudar a visualizar qué trabajo queda por realizar

## Uso
Cada tarea o idea debe documentarse en un archivo `.txt` separado con un nombre descriptivo. Por ejemplo:
- `mejoras-graficas.txt`
- `sistema-inventario.txt`
- `optimizacion-red.txt`

## Relación con otras carpetas
- **contexto/:** Información actual y estable del proyecto
- **tareas/:** Ideas y trabajo pendiente futuro
- **changelog/:** Historial de cambios realizados

## Notas importantes
- Esta carpeta es para trabajo futuro, no para tareas inmediatas
- Las ideas aquí registradas no comprometen implementación
- Sirve como "banco de creatividad" para futuros ciclos de desarrollo
