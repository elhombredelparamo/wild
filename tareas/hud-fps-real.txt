# HUD - MOSTRAR FPS REALES

## Fecha
2026-03-06 - 11:59

## Descripción
Implementar contador de FPS (Frames Per Second) en el HUD para monitorizar rendimiento del juego en tiempo real.

## Características propuestas
- **Contador visual:** Mostrar FPS actuales en esquina de pantalla
- **Colores dinámicos:** 
  - Verde: 60+ FPS (óptimo)
  - Amarillo: 30-59 FPS (aceptable)
  - Rojo: <30 FPS (problemas de rendimiento)
- **Promedio móvil:** Mostrar FPS promedio de últimos segundos
- **Ocultable:** Opción para mostrar/ocultar el contador

## Integración con sistemas existentes
- **HUD actual:** Añadir elemento UI a interfaz existente
- **GameSettings.cs:** Opción para activar/desactivar contador FPS
- **Rendimiento:** Conexión con motor de Godot para obtener FPS
- **Menú opciones:** Configuración de visibilidad y posición

## Consideraciones técnicas
- **Actualización频率:** Refrescar cada 0.5-1.0 segundos (no cada frame)
- **Posición:** Esquina superior derecha (no interferir con juego)
- **Tamaño:** Texto pequeño pero legible
- **Rendimiento:** Mínimo impacto en FPS al mostrarlo

## Variables configurables
- **Visibilidad:** Activado/desactivado
- **Posición:** Esquina seleccionable
- **Transparencia:** Nivel de opacidad del texto
- **Umbral colores:** Configurar límites para colores

## Beneficios
- **Debug:** Identificar problemas de rendimiento
- **Optimización:** Ver impacto de cambios gráficos
- **Usuario:** Información de rendimiento en tiempo real

Alta - Herramienta esencial para optimización y debug
